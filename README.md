# sexy-roster-bot
Create readable BattleScribe's HTML roster via Telegram

Based on r0tational's [killteam-battlescribe](https://github.com/r0tational/killteam-battlescribe) CSS

## How to use
1. Start chat with bot: tg://resolve?domain=zhekud_sexyroster_bot
2. Create roster in BattleScibe
3. Share your roster with bot in HTML format
4. Bot will send edited HTML back to you
5. Open and enjoy ;)
